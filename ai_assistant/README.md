# 🤖 AI Assistant для Windows

Персональный ИИ-ассистент с голосовым управлением, работающий на базе Qwen (Alibaba Cloud).

## ✨ Возможности

1. **Ответы на вопросы** - текстовые ответы на любые вопросы
2. **Открытие приложений** - быстрый поиск через меню Пуск или по заданным путям
3. **Открытие сайтов** - запуск в Microsoft Edge
4. **Анализ экрана** - просмотр и анализ содержимого экрана через скриншоты
5. **Работа с Office** - создание текста, таблиц, презентаций в Word, Excel, PowerPoint
6. **Рисование в Paint** - управление рисованием
7. **Управление мышью** - клики, перемещение, перетаскивание

## 🎨 Дизайн

Современный интерфейс в стиле **"Матовое стекло" (Glassmorphism)**:
- Полупрозрачные элементы
- Неоновые акценты
- Светящийся индикатор статуса
- Адаптивный размер окна

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd ai_assistant
pip install -r requirements.txt
```

### 2. Настройка API ключа

Получите API ключ на [DashScope Console](https://dashscope.console.aliyun.com/):

**Вариант 1: Через переменную окружения (рекомендуется)**
```bash
set QWEN_API_KEY=ваш_ключ_здесь
```

**Вариант 2: В файле config/settings.py**
```python
QWEN_API_KEY = "ваш_ключ_здесь"
```

### 3. Запуск

```bash
python main_entry.py
```

## ⌨️ Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Win + Shift + Enter` | Открыть/закрыть окно ассистента |
| `Enter` (в поле ввода) | Отправить сообщение |
| `Escape` | Закрыть окно |

```

│   ├── __init__.py

│   ├── __init__.py

├── main_entry.py           # Точка входа
└── requirements.txt        # Зависимости
```

## 🔧 Архитектура

Проект использует паттерн **Page Object Model**:

- **AIEngine** - взаимодействие с Qwen API
- **SystemController** - управление приложениями, мышью, экраном
- **UIController** - управление интерфейсом
- **HotkeyManager** - глобальный перехват комбинаций клавиш
- **AssistantWindow** - Page Object главного окна

## ⚠️ Важные замечания

1. **Windows 10/11** - приложение разработано для Windows
2. **Python 3.8+** - требуется современная версия Python
3. **API ключ** - без ключа AI функции не работают
4. **Права доступа** - для управления некоторыми приложениями могут потребоваться права администратора
5. **Microsoft Edge** - должен быть установлен для открытия сайтов

## 🔑 Получение API ключа Qwen

1. Перейдите на https://dashscope.console.aliyun.com/
2. Зарегистрируйтесь или войдите
3. Создайте новый API ключ
4. Скопируйте ключ и установите в настройках
